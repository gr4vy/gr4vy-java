# Frequency

Indicates the frequency unit for the subscription. Allowed values are: `WEEKLY`, `MONTHLY`, `QUARTERLY`, `SEMI_ANNUAL`, `ANNUAL`.


## Values

| Name          | Value         |
| ------------- | ------------- |
| `WEEKLY`      | WEEKLY        |
| `MONTHLY`     | MONTHLY       |
| `QUARTERLY`   | QUARTERLY     |
| `SEMI_ANNUAL` | SEMI_ANNUAL   |
| `ANNUAL`      | ANNUAL        |
